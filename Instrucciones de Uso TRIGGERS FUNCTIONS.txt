Intrucciones de Uso
Requisitos

MariaDB 10.x o MySQL 8.x
Cliente SQL: HeidiSQL, MySQL Workbench o DBeaver

Paso 1 — Ejecutar el script principal
Copiar y ejecutar el script de la sección README.MD "DATABASE LAWFIRM". Crea la base de datos, todas las tablas, índices y datos de prueba

Paso 2- Antes de ejecutar los triggers y functions
Copiar y ejecutar los cinco CREATE TABLE IF NOT EXISTS al inicio del script antes de crear cualquier trigger o function

Paso 3 — Ejecutar los trigger
Copiar y ejecutar cada trigger por separado desde DELIMITER $$ hasta DELIMITER ;.

En HeidiSQL: seleccionar el bloque completo y presionar la tecla F9

Paso 4 — Ejecutar las functions
Copiar y ejecutar cada función por separado desde DELIMITER $$ hasta DELIMITER ;, igual que los triggers

Paso 5 — Probar
